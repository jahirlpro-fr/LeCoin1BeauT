# LE COIN 1 BEAUT - Site E-Commerce de Luxe

Site e-commerce premium pour la vente de produits de beauté et parfums, inspiré des plus grandes maisons de cosmétiques (Nocibé, Aesop, Chanel Beauty).

## 🎨 Direction Artistique

**Évocation Design :** Élégance botanique minimaliste — sophistication d'Aesop avec l'accessibilité de Chanel Beauty.

**Palette de Couleurs Premium :**
- **Noir Profond** `#0A0A0A` — Sophistication, ancrage visuel
- **Crème Ivoire** `#FDFBF7` — Douceur luxueuse (plus chaleureux que blanc pur)
- **Or Champagne** `#D4AF37` — Accents précieux sur CTAs et détails
- **Gris Perle** `#F5F5F0` — Surfaces secondaires, cartes produits
- **Rose Poudré** `#F4E8E0` — Touches subtiles beauté (hover, badges)

**Typographie :**
- **Titres :** Cormorant Garamond (serif élégant, intemporel)
- **Corps :** Montserrat (sans-serif moderne, excellente lisibilité)

## 🗄️ Architecture Base de Données (Supabase)

### Tables Principales
- **categories** : Organisation catalogue (Parfums, Maquillage, Soins, etc.)
- **products** : Produits avec prix, stock, descriptions, images
- **product_variants** : Variantes (teintes, formats, tailles)
- **cart_items** : Panier utilisateur (persistance)
- **orders** : Commandes clients avec statuts
- **order_items** : Détails articles commandés
- **addresses** : Adresses livraison/facturation clients
- **profiles** : Profils utilisateurs (extension auth.users)

### RLS (Row Level Security)
- **Politiques publiques** : Lecture produits/catégories accessible à tous
- **Politiques authentifiées** : Panier, commandes, adresses limitées à l'utilisateur connecté
- **Admin** : Accès complet gestion via roles (à implémenter Phase 5)

## 📦 Installation & Démarrage

```bash
# Installation dépendances
npm install

# Variables d'environnement (.env.local)
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key

# Développement local
npm run dev

# Build production
npm run build

# Démarrage production
npm start
```
